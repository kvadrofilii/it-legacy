# Быстрые команды для консоли (Alias for shell)

Добавляются в файл .bash_profile или .zprofile в каталоге юзера

```shell
alias gs='git status '
alias ga='git add '
alias gb='git branch '
alias gc='git commit'
alias gco='git checkout '
```
